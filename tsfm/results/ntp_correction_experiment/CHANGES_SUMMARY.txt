NTP CORRECTION EXPERIMENT - CHANGES SUMMARY

Date: 2025-10-11

CHANGES IMPLEMENTED:

1) NEW CORRECTION METHOD: "advance_absolute"
   ------------------------------------------
   Location: chronotick_inference/real_data_pipeline.py

   Key Concept:
   - Instead of correcting the POINT error at t_NTP (the "height")
   - Corrects the ACCUMULATED error over the interval (the "area")
   - Formula: accumulated_error = error × interval_duration / 2

   Rationale:
   - Represents total time-integration of drift over the interval
   - For linear drift from 0 to E over duration Δt:
     accumulated_error = ∫[0→Δt] (E/Δt)×t dt = E×Δt / 2
   - This is the "area under the error curve" - a holistic measure
   - Penalizes longer drift periods more heavily

   Differences from 'advanced':
   - 'advanced': Distributes point error E using uncertainty weighting
   - 'advance_absolute': Distributes accumulated error (E × Δt / 2)

   Usage:
     method: advance_absolute
